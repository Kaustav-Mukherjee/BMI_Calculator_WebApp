# BMI Calculator WebApp

A premium, interactive BMI Calculator built with modern web technologies.

## Features
- Real-time BMI calculation based on weight (lbs) and height (in).
- Dynamic health categorization matching professional scales.
- Interactive BMI Gauge visualization.
- Personalized health recommendations.
- Sleek, responsive UI with modern animations.

## Development
This project was developed using:
- **Lovable.dev**: For the initial rapid application prototyping and UI generation.
- **Antigravity (by Google DeepMind)**: For logic refinement, bug fixing (specifically aligning the BMI scale with python-based logic), and project maintenance.

## Tech Stack
- React
- TypeScript
- Tailwind CSS
- Lucide React
- Vite

## How to Run
1. Clone the repository.
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
